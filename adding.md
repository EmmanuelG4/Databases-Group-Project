# How to add to the github:
1. Start with an updated main branch:
```bash
git checkout main
git pull origin main
```

2. Create a new feature branch:
```bash
git checkout -b your-next-feature
```

3. Make your changes and commit them:
```bash
git add .
git commit -m "Implement your next feature"
```

4. Push your feature branch to GitHub:
```bash
git push origin your-next-feature
